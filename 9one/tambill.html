ขั้นตอน

  1. ไปที่ supabase.com/dashboard แล้วสร้างโปรเจกต์ใหม่ (ฟรี) สำหรับร้านของคุณ
  2. เปิด SQL Editor ในโปรเจกต์นั้น แล้ววางคำสั่ง SQL ด้านล่างนี้ กด Run
  3. ไปที่ Project Settings → API คัดลอก Project URL และ anon public key มาวางด้านล่าง
