Master Database Setup

สคริปต์ติดตั้งโครงสร้างฐานข้อมูลเริ่มต้นใน Supabase SQL Editor (รันครั้งเดียวตอนตั้งโปรเจกต์)

ขั้นตอนติดตั้งระบบใหม่

1. รันสคริปต์ SQL ด้านล่างนี้ใน Supabase SQL Editor

2. ไปที่ Authentication → Providers → Email แล้วปิด "Confirm email" หากต้องการให้สมัครแล้วเข้าใช้งานได้ทันที (หรือเปิดไว้เพื่อความปลอดภัยที่มากขึ้น)

3. สมัครสมาชิกร้านค้าแรกผ่านหน้า Login ตามปกติ (จะได้สิทธิ์ STORE_ADMIN)

4. รันคำสั่ง "เลื่อนสิทธิ์เป็น Super Admin" ด้านล่างสุด โดยแก้อีเมลเป็นของคุณ เพื่อให้บัญชีแรกเป็นผู้ดูแลระบบสูงสุด

ลิงก์หน้าแจ้งยอด (สำหรับลูกค้า)

URL ของไฟล์ที่ลูกค้าเปิดดูบิล/แนบสลิป — ใช้สร้างลิงก์แชร์อัตโนมัติในแท็บ "จัดการบิล"

โซนอันตราย (Danger Zone)

หากต้องการเปลี่ยนการเชื่อมต่อไปยัง Project อื่น กรุณากดปุ่มด้านล่างเพื่อลบค่า Config ปัจจุบันทิ้ง

เพิ่มลูกค้าใหม่ (Manual)

สร้าง Tenant สำรองในระบบ — ยังไม่มีบัญชีผู้ใช้ ลูกค้าต้องไปสมัครสมาชิกเองที่หน้า Login แล้วแจ้งให้แอดมินผูก Tenant นี้ให้ภายหลัง (แก้ไขได้ที่ตาราง Master DB > profiles)
